# 🏭 Sistema EHSPro - Segurança do Trabalho

> **Sistema completo de gestão QSSMA (Qualidade, Saúde, Segurança e Meio Ambiente)**

![Demo](demo.gif)

## 🎯 Sobre o Projeto

O **EHSPro** é uma solução moderna e completa para gestão de segurança do trabalho e compliance empresarial. Desenvolvido com tecnologias de ponta, oferece uma interface intuitiva e funcionalidades robustas para:

- 📋 **Gestão de Documentos** - Controle completo de certificados e documentações
- 🦺 **Controle de EPI** - Estoque, entregas e rastreamento biométrico
- 👥 **Gestão de Colaboradores** - Perfis, permissões e histórico completo
- 📊 **Dashboard Inteligente** - KPIs e relatórios em tempo real
- 📱 **Interface Responsiva** - Acesso via desktop, tablet e mobile

## 🚀 Tecnologias

- **Frontend:** Next.js 14, React 18, TypeScript
- **Estilização:** Tailwind CSS, Lucide Icons
- **Autenticação:** Sistema próprio com controle de sessão
- **Biometria:** Integração com dispositivos de captura
- **PDF:** Geração automática de documentos e relatórios
